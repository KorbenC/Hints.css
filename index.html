Hints.css

A simple minimalistic CSS tooltip for modern browsers

Hints.css is a tooltip library based on the power of data-* attribute, pseudo elements, content property, CSS3 transforms and transitions in modern browsers.

Any element on your page which needs to have a tooltip has to be given at least one of the position classes: hint--top, hint--bottom, hint--left, hint--right to position the tooltip.

Getting Started

After you download hints.css, move hints.css to your root's CSS directory. Next, include hints CSS file inside of your tags:

	 <head>
		<link href="css/hints.css" rel="stylesheet">
	 </head>

Adding hints to any element on your page extremely simple. Below is an example of a hint in a p tag.

	 <p class="hint hint--top" data-hint="A hint inside a p tag.">Hello World, hover over me!</p>

Hello World, hover over me!

Adding a tooltip to an link:

	<a class="hint hint--top" data-hint="Follow me on Twitter!" href="#">yourlink</a>
Twitter

Adding a tooltip to an image:

	<span class="hint hint--right" data-hint="Follow me on Facebook!"><img src="yourimage.png"></span>
